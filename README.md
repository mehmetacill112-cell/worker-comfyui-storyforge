# worker-comfyui-storyforge

Custom RunPod `worker-comfyui` image for the storyforge kids YouTube pipeline.
Adds `comfyui-ltxvideo` (LTXV* nodes) and `comfyui-videohelpersuite`
(`VHS_VideoCombine`) on top of `runpod/worker-comfyui:5.8.5-base`.

Pull: `ghcr.io/mehmetacill112-cell/worker-comfyui-storyforge:latest`
